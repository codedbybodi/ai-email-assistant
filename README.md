# AI-Email-Assistant

A python tool that reads any email and generates 3 professional reply options using Google Gemini AI.

## What it does
- Paste any received email
- Get 3 reply options instantly (formel, friendly, concise)
- Powered by Gemini 3 Flash API

## How to run it 
1. Install dependencies: `pip install flask google-genai`
2. Add your Gemini API key
3. Run: `Python email-assistant.py`
4. Open `https://localhost:5000`

## Tech stack
- Python
- Flask
- Google Gemini API
